# Canvas-AI
A online art evaluator and marketplace for all art fanatics.

## Tech Stack
1. Nextjs 
2. Supabase (f/ db)
3. Clerk (f/ auth)
4. Drizzle (f/ orm) 
5. Vercel (f/ hosting)
6. Tailwind (f/ css)
7. Typescript (f/ types)


## Getting Started Locally
*NOTE: If you don't have pnpm installed, use npm instead*
1. Clone repo locally 
```bash
git clone <repo-url>
```
2. Install Dependencies
```bash
pnpm install
```
3. Run dev script  
```bash
pnpm dev
```
